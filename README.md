# gen-lgtm

Convert your favorite images to LGTM images.

## Usage

```bash
$ gen-lgtm --src src.gif --dist dist.gif --loop 5
```

If 0 is set to `loop`, this creates an infinite loop.  
The default value of `loop` is -1, which means it loops once.
